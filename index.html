战略框架

经典的战略框架用于商业分析

波特五力模型

分析行业竞争环境

如何使用:

  1. 识别供应商及其议价能力
  2. 分析买方议价能力
  3. 评估新进入者威胁
  4. 评估替代品威胁
  5. 分析竞争激烈程度

SWOT分析

分析优势、劣势、机会、威胁的战略规划技术

如何使用:

  1. 列出内部优势
  2. 识别内部劣势
  3. 寻找外部机会
  4. 识别外部威胁
  5. 基于分析制定策略

PEST分析

宏观环境分析框架

如何使用:

  1. 分析政治因素
  2. 评估经济因素
  3. 评估社会因素
  4. 审查技术因素

BCG矩阵

产品组合分析工具

如何使用:

  1. 识别明星产品（高增长、高份额）
  2. 找到现金牛产品（低增长、高份额）
  3. 识别问题产品（高增长、低份额）
  4. 识别瘦狗产品（低增长、低份额）

思维框架

强大的结构化思维和问题解决框架

第一性原理思维

将复杂问题分解为基本要素

如何应用:

  1. 识别并定义当前假设
  2. 将问题分解为基本原则
  3. 从头开始创建新解决方案
  4. 测试和验证新方法

MECE原则

相互独立、完全穷尽

如何应用:

  1. 将问题分解为相互独立的类别
  2. 确保覆盖所有可能性
  3. 避免类别之间的重叠
  4. 用于全面问题分析

SCAMPER技术

创新的创造性思维工具

如何应用:

  1. 替代 - 什么可以被替换？
  2. 合并 - 什么可以合并？
  3. 适应 - 什么可以调整？
  4. 修改 - 什么可以改变？
  5. 另用 - 新的应用？
  6. 消除 - 什么可以移除？
  7. 反转 - 什么可以反转？

六顶思考帽

平行思考方法，提升决策质量

如何应用:

  1. 白帽 - 事实和信息
  2. 红帽 - 情绪和感受
  3. 黑帽 - 谨慎和批评
  4. 黄帽 - 利益和乐观
  5. 绿帽 - 创造性和替代方案
  6. 蓝帽 - 过程控制
